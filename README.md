# Skills

Skills are folders of instructions, scripts, and resources that Claude loads dynamically to improve performance on specialized tasks. Skills teach Claude how to complete specific tasks in a repeatable way, whether that's creating documents with your company's brand guidelines, analyzing data using your organization's specific workflows, or automating personal tasks.

This repository is where Firely publishes reusable skills that capture practical FHIR workflows, validation patterns, and implementation guidance. The goal is to make these skills easy to discover, easy to reuse, and easy to improve in the open.

## Repository Structure

- `skills/`
Contains published skills. Each skill has its own folder with a `SKILL.md` and optional supporting references.
- `template/`
Base structure for creating new skills.

## Disclaimer

These skills are provided as practical guidance and examples. They are not a substitute for formal regulatory, legal, or security review. Always validate outputs against your organization's policies, clinical safety requirements, and applicable standards.
